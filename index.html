JVM (1)-JVM Memory Area

Posted on 2021-12-02 Edited on 2021-12-04 In JVM

Overview

Java is the well-known language that has the slogan вҖңwrite once, run everywhereвҖқ, and the exist of JVM is exactly the reason why this slogan can be achieved. Specifically, when we write and run the Java code, the .java file is ultimately translated into bytecode, and then be loaded in the JVM to be implemented. This means that as long as we have a JVM in our machine, we can run the compiled JAVA file whatever the OS we are using.

JVM divides the memory into different parts to better manage it, we can say the the process that JVM load and create a Class/Instance is exactly the process that it put data into different parts of memories.

There are FIVE areas in JVM memory, with 2 of them are shared by threads (Heap, Method Area) and 3 of them (Stack, Native Method Stack, Program Counter) are thread-private.

вҖӢ йҖ’еҪ’ж—¶з”ұдәҺйҮҚеӨҚи°ғз”Ёж–№жі•иҮӘиә«пјҢжҜҸж¬Ўи°ғз”Ёж–№жі•ж—¶зҡҶйңҖеңЁеҶ…еӯҳж ҲдёӯеҲҶй…Қз©әй—ҙд»ҘдҝқеӯҳеҸӮж•°пјҢиҝ”еӣһең°еқҖеҸҠдёҙж—¶еҸҳйҮҸ еӣ жӯӨйңҖеӨ§йҮҸз©әй—ҙпјҢиҖҢеҺӢж Ҳеј№ж ҲзҡҶйңҖиҰҒж—¶й—ҙгҖӮ

вҖӢ

вҖӢ еҶҷйҖ’еҪ’д»Јз ҒеҲҮеҝҢдәәи„‘жЁЎжӢҹз”өи„‘жҖқиҖғиҝӣж ҲеҮәж Ҳз»ҶиҠӮгҖӮ

вҖӢ еҸӘйңҖзЎ®и®ӨдёүзӮ№пјҡ

вҖӢ з»ҲжӯўжқЎд»¶ + еҮҪж•°еҠҹиғҪ + еҮҪж•°ж–№зЁӢпјӣ

вҖӢ жӯӨеӨ–жҖқиҖғеҮҪж•°ж–№зЁӢж—¶пјҢд№ҹйңҖжіЁж„Ҹз»ҲжӯўжқЎд»¶дёӯзҡ„иҝ”еӣһеҖјжҳҜд»Җд№ҲгҖӮ

вҖӢ

вҖӢ еҗҢж—¶пјҢ еҲ«еӨӘжӯ»жқҝзҡ„и§үеҫ— з»ҲжӯўжқЎд»¶дёҖе®ҡиҰҒ вҖңеҗҢж—¶еҢ…еҗ«дә’ж–ҘжқЎд»¶зҡ„дёӨж–№вҖқпјҢ дҫӢеҰӮ offer 13. еҸӘйңҖеҲӨж–ӯеҪ“еүҚж јеӯҗдёҚеҸҜйҖүж—¶пјҢreturn 0еҚіеҸҜгҖӮ ж— йңҖеҢ…еҗ« еҪ“еүҚж јеӯҗжҳҜжңҖеҗҺдёҖ дёӘеҸҜйҖүзҡ„ж—¶иҝ”еӣһ1пјҢ и§ӮеҜҹеҮҪж•°ж–№зЁӢеҚіеҸҜзҗҶи§ЈгҖӮ

еҠЁжҖҒи§„еҲ’

вҖӢ зү№зӮ№пјҡ

вҖӢ пјҲ1пјүжҹҗдёӘй—®йўҳеҸҜд»ҘеҲҶи§ЈжҲҗе°Ҹй—®йўҳ пјҢе№¶дё”жүҖй—®дёәжұӮжңҖдјҳи§ЈпјҲе…ідәҺжҳҜеҗҰиҰҒжұӮжңҖдјҳи§ЈдёҚз»қеҜ№ пјүгҖӮ

вҖӢ пјҲ2пјүжҠҠе°Ҹй—®йўҳзҡ„жңҖдјҳи§Јз»„еҗҲиө·жқҘиғҪеҫ—еҲ°ж•ҙдҪ“жңҖдјҳи§ЈпјҢеҲҷйҖҡеёёеҸҜз”ЁеҠЁжҖҒи§„еҲ’гҖӮ

вҖӢ пјҲ3пјү е°Ҹй—®йўҳд№Ӣй—ҙжңүйҮҚеҸ зҡ„еӯҗй—®йўҳпјҲеҚіеҰӮжһңз”ЁйҖ’еҪ’еҲҷйңҖиҰҒз®—еҫҲеӨҡйҮҚеӨҚзҡ„еҖјпјүгҖӮ

вҖӢ еҲҶжһҗеҠЁжҖҒи§„еҲ’ж—¶зҡ„жҖқиҖғж–№ејҸдёҺйҖ’еҪ’зұ»дјјпјҢеӣ дёәйғҪжҳҜеӨ§й—®йўҳеҲҶи§Је°Ҹй—®йўҳгҖӮ

вҖӢ еҠЁжҖҒи§„еҲ’еҲҷдёәиҮӘдёҠиҖҢдёӢ ж–№ејҸжҖқиҖғи§ЈдҪ“жҖқи·ҜгҖӮиҖҢз”Ёд»Јз Ғе®һзҺ°ж—¶еҲҷз”ЁиҮӘдёӢиҖҢдёҠ зҡ„ж–№ејҸи§ЈеҶій—®йўҳпјҢеҗҢж—¶еҲ©з”Ёж•°жҚ®еӯҳеӮЁе°Ҹй—®йўҳзҡ„и§Јз„¶еҗҺз»„еҗҲе°Ҹй—®йўҳзҡ„и§Јд»Ҙеҫ—еҲ°ж•ҙдёӘй—®йўҳзҡ„и§ЈпјҢ зҶҹз»ғеҗҺеҸҜжҠӣејғж•°з»„пјҢд»…з”ЁеҮ дёӘеҸҳйҮҸ еӯҳеӮЁжңүз”Ёзҡ„и§Ј

иҙӘеҝғ

вҖӢ иӢҘжҹҗдёӘй—®йўҳз”ЁеҠЁжҖҒи§„еҲ’зҡ„жҖқи·ҜжҖқиҖғд»ҘеҗҺпјҢ еҸ‘зҺ°еҲҶи§Јеӯҗй—®йўҳж—¶еӯҳеңЁжҹҗз§Қзү№е®ҡжқЎд»¶иғҪдҪҝеҫ—з»“жһңдёҖе®ҡдёәжңҖдјҳи§ЈпјҢ еҲҷжӯӨж—¶еә”жҖқиҖғиҙӘеҝғз®—жі•

еӣһжәҜ

еҸҜзҗҶи§ЈдёәжңүеүӘжһқзҡ„з©·дёҫпјҢе…¶д»Һи§ЈеҶій—®йўҳжҜҸдёҖжӯҘзҡ„еӨҡдёӘйҖүйЎ№йҮҢзі»з»ҹзҡ„йҖүжӢ©дёҖдёӘеҸҜиЎҢзҡ„ж–№жЎҲгҖӮ

йҖӮеҗҲз”ЁдәҺи§ЈеҶіжңүеӨҡдёӘжӯҘйӘӨз»„жҲҗпјҢдё”жҜҸдёӘжӯҘйӘӨжңүеӨҡдёӘйҖүйЎ№зҡ„й—®йўҳгҖӮ

еӣһжәҜжі•и§ЈеҶій—®йўҳзҡ„иҝҮзЁӢпјҢеҸҜд»Ҙиҝ‘дјјзҗҶи§ЈдёәвҖңйҒҚеҺҶвҖқдёҖйў—ж ‘зҡ„иҝҮзЁӢгҖӮ ж ‘дёӯзҡ„жҜҸдёҖдёӘlevel еҚідёҖдёӘжӯҘйӘӨпјҢжҜҸдёӘnodeеҚідёҖдёӘйҖүйЎ№гҖӮ

еҪ“йҒҚеҺҶиҮідёҖдёӘиҠӮзӮ№ж—¶пјҢеҸ‘зҺ°з»“жһңе·ІдёҚеҸҜиғҪж»Ўи¶іжқЎд»¶пјҢеҲҷеҸҜж”ҫејғиҜҘиҠӮзӮ№иҝ”еӣһиҮідёҠдёҖеұӮиҠӮзӮ№гҖӮ

еҗҢзҗҶпјҢиӢҘеҸ‘зҺ°дёҖдёӘиҠӮзӮ№зҡ„жүҖжңүеӯҗиҠӮзӮ№еқҮдёҚж»Ўи¶іжқЎд»¶ж—¶пјҢеҲҷеҗҢж ·иҝ”еӣһиҮідёҠдёҖеұӮиҠӮзӮ№гҖӮ

иҖҢиҝ”еӣһиҮідёҠдёҖеұӮиҠӮзӮ№ж—¶пјҢиӢҘеҸ‘зҺ°дёҠдёҖеұӮдёӯжүҖжңүиҠӮзӮ№еқҮе·Іиў«йҒҚеҺҶиҝҮпјҢеҲҷеҶҚж¬Ўиҝ”еӣһиҮідёҠдёҖеұӮиҠӮзӮ№гҖӮ

еҰӮжһңжүҖжңүиҠӮзӮ№пјҲеҚіи·Ҝеҫ„пјүеқҮе·ІйҒҚеҺҶдё”ж— з¬ҰеҗҲжқЎд»¶зҡ„и·Ҝеҫ„пјҢеҲҷr

еҜ№дәҺеңЁдәҢз»ҙж•°з»„дёӯжҗңзҙўи·Ҝеҫ„зҡ„й—®йўҳпјҢеҰӮиҝ·е®«жҲ–жЈӢзӣҳпјҢд»ҘеҸҠжҗңзҙўеҸҜиЎҢи§Јзҡ„й—®йўҳпјҢеҲҷйҖҡеёёеҸҜз”ЁеӣһжәҜжі•гҖӮ иҖҢеӣһжәҜжі•йҖҡеёёеҸҜз”ЁйҖ’еҪ’зҡ„ж–№ејҸжұӮи§ЈгҖӮ йҖ’еҪ’еҸҲеҸҜз”Ёж ҲжқҘе®һзҺ°гҖӮ
